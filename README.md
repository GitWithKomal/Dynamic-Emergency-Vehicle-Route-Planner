# Emergency Route Finder

A system that helps emergency vehicles find the fastest route using real-time map data.

## Features
- Dynamic route planning
- Traffic-aware navigation
- Emergency priority routing

## Tech Stack
- HTML
- JavaScript
- Python
- Mappls API

## Mappls API Key Setup

This project uses the Mappls API for map and routing services.

To run the project, you need your own Mappls API key.

1. Go to the Mappls developer portal.
2. Create an account and generate an API key.
3. Replace the API key in the following file:

Frontend/script.js

Example:

const MAPPLS_API_KEY = "YOUR_API_KEY_HERE";
